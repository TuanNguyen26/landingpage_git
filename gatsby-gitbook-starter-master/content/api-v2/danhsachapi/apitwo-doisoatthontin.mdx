### Đối soát thông tin

Thông tin đối soát có thể được gửi kèm theo parameter `check_sum`, và được áp dụng trên tất cả các API

**Params**:

| Key         | Value             | Mô tả                             |
| ----------- | ----------------- | --------------------------------- |
| `check_sum` | `120EA8A25E5D487` | Trường thông tin dữ liệu đối soát |
